# 🔱 Ven Hub - Blooket Utility (2025/2026)
An advanced, draggable GUI for Blooket game mode testing and global utility. Inspired by the Glixzzy repository.

## ✨ Features
- **Modern UI:** Draggable Black & Gold interface.
- **Global Actions:** Instant Token/XP increments and Blook unlocks.
- **Mode Specifics:** Specialized modules for Gold Quest, Cafe, Factory, and more.
- **Live Updates:** Fetches latest script logic directly from source.

## 🚀 How to Use
1. Copy the code from `venhub.js`.
2. Create a new bookmark in your browser.
3. In the URL/Address field, type `javascript:` and then paste the code.
4. Click the bookmark while on a Blooket page.

## ⚠️ Disclaimer
This project is for **educational purposes only**. It is intended to demonstrate how JavaScript interact with web elements. Using this on live games may violate Blooket's Terms of Service and result in an account ban. Use at your own risk.
